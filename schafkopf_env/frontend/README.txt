Get started with frontend testing (pc)

1. (console) start backend 

2. (console) cd frontend
    python -m http.server 5500

3. (browser) http://localhost:5500


Get started with frontend testing (smartphone)

1. (console) ifconfig | grep "inet " | grep -v 127.0.0.1

2. (console) start backend 

3. (console) cd frontend
    python -m http.server 5500

4. (browser) http://<DEINE_IP>:5500
